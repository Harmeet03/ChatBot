# RoboChat (ChatBot)
ChatBot is a full-stack AI-powered chatbot application built using the MERN stack for authentication and user management, combined with a Python Flask service that connects to the Hugging Face AI inference API. 

The project demonstrates a real-world multi-layer architecture where frontend, backend, and AI services communicate through REST APIs.

## ✨ TECHNOLOGIES
- `React.js`
- `Node.js`
- `Express.js`
- `MongoDB`
- `Python`
- `Flask`
- `Hugging Face Inference API`
- `JavaScript` (ES6)
- `CSS3`
- `Vite`

## 🚀 FEATURES
- Secure user authentication using JWT
- AI-generated responses via Hugging Face API
- Full-stack API communication (React → Flask → HuggingFace API's AI → Back to Flask → React)

## 📍 THE PROCESS
I built this project to understand how AI services can be integrated into a secure full-stack application. The goal was to implement authentication using MERN, create a middleware layer using Flask, and connect it to a third-party AI inference API (Hugging Face). 

This helped me learn API chaining, authentication flow, and multi-service backend architecture.

## 🚦 RUNNING THE PROJECT

- Clone the repository  
- Install dependencies: `npm install`
- Run frontend: `npm run dev`
- Run Node backend: `node server.js`  
- Run Flask server: `python app.py`  
- Open `http://localhost:5173` in your browser
